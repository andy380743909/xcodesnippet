# xcodesnippet
*A command-line interface for installing Xcode Snippets*

## Installation

```
$ gem install oss_xcodesnippet
```

## Usage

#### Sourcefile.swift

```swift
---
title: "Hello, World!"
summary: "Prints 'Hello World'"
completion-scope: Function or Method
---

println("Hello, World!")
```

#### Terminal Command

```
$ oss_xcodesnippet install path/to/source.m
```
